# Title 标题

`pa-title` 用于展示带有装饰性下划线的标题组件。

## 基础使用

<demo src="./base.vue"></demo>

## Props

| 字段    | 描述           | 类型                                                     | 默认值      |
| ------- | -------------- | -------------------------------------------------------- | ----------- |
| line    | 是否使用下划线 | `boolean`                                                | `true`      |
| tips    | 提示信息       | `string`                                                 | - |
| padding | 内边距方向     | `Array<'top' \| 'left' \| 'bottom' \| 'right' \| 'all'>` | - |
