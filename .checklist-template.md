# 组件合规性排查任务模板

## 项目背景
- 项目路径：/root/Butler-AI-Website
- 组件名：{组件名}
- 组件路径：/root/Butler-AI-Website/src/package/components/pa-{组件名}/
- 文档路径：/root/Butler-AI-Website/document/PancakeUI_Doc/{组件名}/
- 排查规则：项目根目录 `checklist.md`

## 排查要求
1. **必须先读取 checklist.md**（使用 `cat /root/Butler-AI-Website/checklist.md`），严格按照文件内容执行
2. 读取组件的所有相关文件：
   - types.d.ts
   - index.scss
   - .vue 文件
   - index.ts
   - 文档 readme.md
   - 示例文件 base.vue
3. 按照 checklist.md 中的 5 个表格逐项排查

## 关键规则
- 所有文件必须使用 Prettier 进行格式化
- Vue 文件代码顺序：`import → const/let变量 → function → 生命周期钩子 → watch`
- ts 文件代码顺序：`import → const/let变量（vue相关）→ const/let其他变量 → function → vue的watch`
- 所有方法必须使用 function 定义，而非 const 定义
- 必须定义属性：id、class、style（可选属性，带?）

**必须输出完整表格**，每项标注 ✅ 或 ❌。格式如下：
